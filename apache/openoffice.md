# Apache Openoffice

## O que é o Projeto?

O Apache OpenOffice é uma suíte de aplicativos de código aberto para escritório que inclui processador de texto, planilha, programa de apresentação, editor de gráficos vetoriais e muito mais. Ele é uma alternativa de software livre ao Microsoft Office e ao LibreOffice, oferecendo uma ampla gama de recursos para criar e editar documentos, planilhas e apresentações[^1].

## Quais são seus objetivos?

O projeto Apache OpenOffice tem como principais objetivos oferecer um conjunto de aplicativos de escritório de código aberto de alta qualidade, manter a compatibilidade com formatos de arquivo populares, garantir a acessibilidade em diversas plataformas, permitir a extensibilidade e promover o desenvolvimento colaborativo de código aberto. Seu objetivo central é proporcionar uma alternativa gratuita ao software proprietário de suíte de escritório, atendendo às necessidades de uma ampla variedade de usuários em todo o mundo[^2].

## Qual sua importância ?

O projeto Apache OpenOffice é importante para a Apache Software Foundation e a comunidade de software livre como um todo, pois representa um esforço significativo de desenvolvimento de software de código aberto que oferece uma suíte de escritório completa e acessível, ajudando a promover a liberdade do software e a opção de escolha para os usuários, reduzindo a dependência de soluções de software proprietário. Além disso, ele demonstra o compromisso contínuo da comunidade de código aberto com a manutenção e aprimoramento de alternativas de software de alta qualidade e de código aberto[^3].

## Status do projeto atualmente

O Apache OpenOffice foi descontinuado devido a uma combinação de fatores, incluindo um estado de desenvolvimento lento e a falta de contribuições regulares por parte da comunidade de desenvolvedores. Isso resultou em um progresso limitado na melhoria do software e na resolução de problemas. Além disso, a disponibilidade de alternativas de suíte de escritório de código aberto, como o LibreOffice, que viu um desenvolvimento mais ativo e maior apoio da comunidade, acabou reduzindo a demanda pelo Apache OpenOffice. Como resultado, a Apache Software Foundation anunciou que não tinha recursos para manter o projeto de forma eficaz, levando à sua descontinuação[^4].

# Aqui estão alguns dos principais componentes do Apache OpenOffice:

Writer
====

 <img src="https://www.openoffice.org/product/pix/writer-big.png" alt="Exemplo do writer" width="400"/> <br />

O Writer é o processador de texto da suíte OpenOffice, que oferece recursos completos para criação e formatação de documentos. Ele inclui assistentes para tarefas padrão, suporte a estilos e formatação avançados, correção automática, conclusão automática e a capacidade de trabalhar com múltiplas línguas. Além disso, permite a criação de documentos complexos, como boletins informativos, e oferece recursos como tabelas de conteúdo, notas avançadas e exportação para diversos formatos, incluindo PDF e HTML. O Writer é compatível com formatos Microsoft Word e OpenDocument, proporcionando flexibilidade na troca de documentos[^5].

 
Cal
====

 <img src="https://www.openoffice.org/product/pix/calc-big.png" alt="Exemplo do Cal" width="400"/> <br />

O Calc é um programa de folha de cálculo intuitivo e fácil de aprender, com um conjunto extenso de funções avançadas. Ele possui a tecnologia DataPilot que facilita a adição de dados brutos de bases de dados empresariais. Permite a criação de fórmulas usando linguagem natural, e possui um gestor de cenários para análise “e se…” e um componente “solucionar” que permite resolver problemas de optimização. Oferece assistentes para utilizar um grande conjunto de funções de cálculo avançadas e permite a aplicação flexível de opções de formatação a células. O suporte multi-utilizador do Calc encoraja o trabalho corporativo em folhas de cálculo. Os documentos podem ser gravados em formato OpenDocument, o novo padrão internacional para documentos de escritório, e em Portable Document Format (.pdf). Além disso, o Calc é compatível com Microsoft Excel, podendo abrir ficheiros .xlsx criados com o Microsoft Office 2007 ou Microsoft Office 2008 para Mac OS X[^6].


Impress
====

 <img src="https://www.openoffice.org/product/pix/impress-big.png" alt="Exemplo do Impress" width="400"/> <br />

O Impress é uma ferramenta excepcional para criar apresentações multimédia com arte 2D e 3D, efeitos especiais, animação e ferramentas de desenho. Ele simplifica a preparação dos materiais e oferece um conjunto completo de vistas para atender às necessidades dos apresentadores e do público. Ele suporta múltiplos monitores, permitindo que os apresentadores vejam algo diferente enquanto mostram os slides num projetor. Além disso, possui um conjunto completo de ferramentas de desenho e diagrama para enriquecer a apresentação. Permite gravar documentos no formato OpenDocument, o novo padrão internacional para documentos de escritório. Ele pode ler todos os seus documentos antigos do Microsoft PowerPoint, ou gravar o seu trabalho em formato Microsoft PowerPoint. Alternativamente, pode usar a funcionalidade interna do Impress para criar versões Flash (.swf) do seu trabalho[^7].


Draw
====

<img src="https://www.openoffice.org/product/pix/draw-big.png" alt="Exemplo do Draw" width="400"/> <br />

O Draw é uma aplicação gráfica poderosa que permite desde um esboço rápido até uma planta complexa. Ele oferece ferramentas para comunicar com gráficos e diagramas e é adequado para desenho técnico, cartazes, entre outros. As ferramentas de desenho mais usadas podem ser 'estacionadas' em qualquer lugar da tela para fácil acesso. O Draw permite manipular objetos, rotacioná-los em duas ou três dimensões e criar imagens quase fotográficas com textura própria, efeitos de luz, transparência, perspectiva, etc. Ele simplifica o trabalho com gráficos de fluxo, organogramas, diagramas de rede, etc., e calcula e mostra automaticamente dimensões lineares à medida que você desenha. Você pode criar sua própria arte e adicioná-la à Galeria de imagens. Os documentos podem ser gravados no formato OpenDocument, o novo padrão internacional para documentos de escritório, e importar gráficos de todos os formatos comuns. Além disso, o Draw pode criar versões Flash (.swf) do seu trabalho[^8].


Math
====

<img src="https://www.openoffice.org/product/pix/math-big.png" alt="Exemplo do Math" width="400"/> <br />

O Math é o editor de fórmulas matemáticas multiplataforma e de código aberto da suíte OpenOffice.org. Suporta múltiplas fontes e pode exportar no formato PDF. Curiosamente, apesar da linguagem interna do Math ser muito parecida com o LaTeX, o Math não importa fórmulas escritas em LaTeX. Pode ser usado isoladamente ou integrado aos demais programas do OpenOffice.org. Para isso, ao editar um documento (por exemplo, um texto criado pelo Writer ou uma célula de uma planilha de cálculo do Calc), pode-se invocar o Math para inserir uma fórmula matemática. No Math, uma fórmula pode ser introduzida de três maneiras: 
* Digitando a fórmula no editor de equações.
* Clicando com o botão direito no editor de equações e selecionando a função num dos vários sub-menus contextuais.
* Selecionando um símbolo da caixa Elementos.
  
É muito útil para criação de equações e fórmulas matemáticas para seus documentos[^9].


Base
====

<img src="https://www.openoffice.org/product/pix/base-big.png" alt="Exemplo do Base" width="400"/> <br />

O Base é um sistema gestor de bases de dados completo, projetado para atender a uma ampla gama de usuários, desde a manutenção de uma coleção pessoal de CDs até a produção de um relatório mensal de vendas em uma empresa. Ele oferece assistentes para ajudar os novos usuários a criar tabelas, pesquisas, formulários e relatórios, além de um conjunto de tabelas predefinidas para localizar bens, clientes, vendas, faturas e muito mais. Para uso pessoal, o Base oferece o motor de banco de dados relacional HSQL, configurado para um único usuário, com os dados armazenados no arquivo do Base, bem como suporte para arquivos planos dBase. Para uso empresarial, o Base oferece suporte para uma grande variedade de motores multiusuário: MySQL, Adabas D, MS Access e PostgreSQL. Além disso, os controladores padrão de suporte para JDBC e ODBC permitem que você se conecte a praticamente qualquer banco de dados existente. Se integra perfeitamente com o restante do conjunto de aplicativos do Apache OpenOffice, fornecendo dados do catálogo de endereços para o correio eletrônico no Writer usando o protocolo padrão LDAP, ou formatos comuns de catálogos de endereços como o Microsoft Outlook, Microsoft Windows e Mozilla, e criando intervalos de dados vinculados nas planilhas do Calc para análise de dados ou como base para gráficos[^10].
